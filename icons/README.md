# 图标说明

此文件夹应该包含以下图标文件：

- `icon16.png` - 16x16 像素（工具栏小图标）
- `icon48.png` - 48x48 像素（扩展管理页面）
- `icon128.png` - 128x128 像素（Chrome Web Store）

## 快速添加图标

### 方式一：使用在线工具生成
1. 访问 https://www.favicon-generator.org/ 或类似网站
2. 上传你的图片
3. 下载不同尺寸的图标
4. 重命名为上面的文件名并放到这里

### 方式二：使用临时图标
你可以先随便找 3 张 PNG 图片，重命名后放到这里，扩展也能正常运行！

### 方式三：使用纯色占位
用任何图片编辑工具创建 3 张纯色 PNG 图片即可。

## 注意

⚠️ **没有图标文件扩展也能运行！** Chrome 会显示默认图标。

只是看起来不够美观而已，功能完全不受影响。
