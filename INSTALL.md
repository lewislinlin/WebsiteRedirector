# 🚀 快速安装指南

## 📂 获取项目文件

从 GitHub 下载或克隆项目：
```bash
git clone https://github.com/YOUR_USERNAME/website-redirector.git
```

或者直接下载 ZIP 解压。

## 🔧 安装步骤（Chrome/Edge 浏览器）

### 1. 准备项目文件夹

把 `WebsiteRedirector` 文件夹放到一个方便的位置，比如：
- 桌面
- 文档文件夹

### 2. 打开扩展管理页面

**Chrome 浏览器：**
- 地址栏输入：`chrome://extensions/`
- 或：菜单 → 扩展程序 → 管理扩展程序

**Edge 浏览器：**
- 地址栏输入：`edge://extensions/`
- 或：菜单 → 扩展 → 管理扩展

### 3. 开启开发者模式

在扩展管理页面右上角，找到并打开 **"开发者模式"** 开关。

### 4. 加载扩展

1. 点击 **"加载已解压的扩展程序"** 按钮
2. 浏览并选择 `WebsiteRedirector` 文件夹
3. 点击"选择文件夹"
4. ✅ 完成！

## 🎯 使用说明

### 第一次使用

1. **点击扩展图标**
   - 在浏览器工具栏找到拼图图标（扩展图标）
   - 点击"网站跳转器"

2. **配置目标网站**
   - 在"目标网站"输入你想跳转到的网站
   - 例如：`https://leetcode-cn.com` 或 `https://weread.qq.com`

3. **添加源网站**
   - 输入你想限制的网站
   - 例如：`douyin.com`, `zhihu.com`, `bilibili.com`
   - 点击"+ 添加网站"
   - 可以继续添加更多

4. **选择跳转模式**
   - 立即跳转：最严格，打开就跳转
   - 5 秒倒计时：给你反悔的机会
   - 计时模式：开发中...

5. **启用跳转器**
   - 打开"启用跳转器"开关
   - 试试看！打开源网站应该会自动跳转

### 临时暂停

真的需要用源网站时：
1. 点击扩展图标
2. 选择暂停时长：5 分钟 / 15 分钟 / 30 分钟
3. 期间可以正常使用
4. 时间到了自动恢复，或点击"立即恢复"

## ❓ 常见问题

### Q: 扩展图标在哪里？
A: 在浏览器工具栏的拼图图标里，点击后可以固定到工具栏。

### Q: 图标不显示怎么办？
A: 没关系，功能不受影响！可以看 `icons/README.md` 添加图标。

### Q: 支持哪些浏览器？
A: Chrome、Edge、Brave 等基于 Chromium 的浏览器都可以。

### Q: 设置会同步吗？
A: 当前版本只保存在本地，不会跨设备同步。

### Q: 会上传我的数据吗？
A: 不会！所有数据都存储在浏览器本地，非常安全。
