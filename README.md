# 🌐 网站跳转器 (Website Redirector)

一个 Chrome/Edge 浏览器扩展，当你访问某些网站时自动跳转到目标网站，防止沉迷网络！

## ✨ 功能特点

- 🎯 **支持多个源网站** - 可以添加多个要限制的网站（抖音、知乎、B站等）
- 🔄 **统一跳转到目标网站** - 所有源网站都会跳转到同一个目标网站
- ⏸️ **临时暂停功能** - 真的需要用时可以暂停 5/15/30 分钟
- ⚡ **多种跳转模式**
  - 立即跳转
  - 5 秒倒计时（给你反悔的机会）
  - 计时模式（即将上线）
- 💾 **本地存储** - 所有设置保存在浏览器本地
- 🎨 **美观界面** - 渐变色设计，简洁易用

## 📦 安装方法

### Chrome/Edge 浏览器（开发者模式）

1. **下载或克隆项目**
   ```
   WebsiteRedirector/
   ├── manifest.json
   ├── background.js
   ├── content.js
   ├── popup.html
   ├── popup.js
   └── icons/
   ```

2. **打开扩展管理页面**
   - Chrome: 访问 `chrome://extensions/`
   - Edge: 访问 `edge://extensions/`

3. **开启开发者模式**
   - 右上角打开"开发者模式"开关

4. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择 `WebsiteRedirector` 文件夹
   - 完成！

## 🚀 使用说明

### 1. 配置目标网站
- 点击浏览器工具栏的扩展图标
- 在"目标网站"输入框中输入你想跳转到的网站
- 例如：`https://www.baidu.com` 或学习网站

### 2. 添加源网站
- 在"添加要限制的网站"输入框中输入网站
- 支持格式：
  - 完整域名：`douyin.com`
  - 带 www：`www.zhihu.com`
  - 完整 URL：`https://www.bilibili.com`
- 点击"+ 添加网站"按钮
- 可以继续添加更多网站

### 3. 选择跳转模式
- **立即跳转**：打开源网站立即跳转（最严格）
- **5 秒倒计时**：显示 5 秒倒计时，可以取消跳转
- **计时模式**：开发中...

### 4. 启用跳转器
- 打开"启用跳转器"开关
- 现在试试看！打开源网站应该会自动跳转

### 5. 临时暂停
- 真的需要用源网站时，点击扩展图标
- 选择暂停时长：5 分钟 / 15 分钟 / 30 分钟
- 暂停期间可以正常使用源网站
- 时间到了自动恢复，也可以点击"立即恢复"

## 💡 使用示例

### 示例 1：防止刷短视频
- **目标网站**：`https://leetcode-cn.com`（力扣）
- **源网站**：`douyin.com`, `kuaishou.com`, `bilibili.com`
- 效果：想刷短视频时自动跳去刷题

### 示例 2：减少社交网络
- **目标网站**：`https://weread.qq.com`（微信读书）
- **源网站**：`zhihu.com`, `weibo.com`, `twitter.com`
- 效果：想看八卦时自动跳去读书

### 示例 3：专注工作
- **目标网站**：你的公司内网或工作文档
- **源网站**：`taobao.com`, `jd.com`, `xiaohongshu.com`
- 效果：想购物时自动跳回工作

## ⚙️ 配置说明

### 网站匹配规则
- 输入 `douyin.com` 会匹配：
  - `douyin.com`
  - `www.douyin.com`
  - `m.douyin.com`
  - 任何子域名

### 数据存储
- 所有设置保存在浏览器本地存储中
- 不会上传到任何服务器
- 清除浏览器数据会重置设置

## 🔧 技术栈

- **Manifest Version**: 3（最新版）
- **语言**: 纯 JavaScript（无框架）
- **兼容**: Chrome 88+, Edge 88+, Firefox（需调整）
- **权限**: 
  - `storage` - 本地存储
  - `tabs` - 标签页管理
  - `alarms` - 定时任务
  - `scripting` - 脚本注入

## 🎨 自定义图标

当前使用的是占位图标，你可以替换为自己的图标：

1. 准备 3 个尺寸的 PNG 图标：
   - 16x16 - 小图标
   - 48x48 - 扩展管理页面
   - 128x128 - Chrome Web Store（可选）

2. 命名为：
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`

3. 放到 `icons/` 文件夹中

## 🚧 注意事项

1. **HTTPS 网站**：扩展可以在所有网站上运行，包括 HTTPS
2. **本地文件**：无法在 `file://` 协议的页面上运行
3. **性能影响**：非常轻量，几乎不影响浏览器性能
4. **隐私安全**：所有数据本地存储，不上传服务器
